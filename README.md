# HR Analytics Data Transformation using Excel Power Query

## Project Overview
This project focuses on cleaning and transforming an HR Analytics dataset using Excel Power Query. The objective is to prepare clean, structured data and create meaningful dashboards for analysis.

## Tools Used
- Microsoft Excel
- Power Query
- PivotTables & Charts

## Data Transformation Steps
- Imported HR Analytics dataset into Excel Power Query
- Removed unnecessary columns
- Handled missing values by removing critical numeric nulls and replacing categorical nulls
- Corrected data types for numeric and date columns
- Created Age Group and Salary Band using conditional columns
- Loaded transformed data into Excel for analysis

## Dashboards Created
- Employee Attrition Analysis
- Salary Band Distribution
- Age Group vs Attrition Analysis

## Files Included
- `task12_data_transformation.xlsx` – Cleaned data and dashboards
- `dashboard_export.pdf` – Exported dashboard visuals
- `transformation_notes.txt` – Summary of Power Query steps

## Outcome
The project provides insights into employee attrition, salary distribution, and workforce demographics using structured and clean HR data.
